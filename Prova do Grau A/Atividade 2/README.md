# Prova do Grau A: Atividade 2
## Autor
- João Pedro Büttenbender da Silva
## Descrição
```
Crie um programa para uma loja que venda livros, CDs e DVDs. O sistema deve permitir a
impressão dos seguintes dados:

• Para livros: nome, preço e autor;
• Para CDs: nome, preço e número de faixas;
• Para DVDs: nome, preço e duração.
• Adicione um atributo que represente o código de barras do produto (é um valor
obrigatório e, portanto, deve ser pedido no construtor);
• Forneça um método que verifica se dois produtos são iguais, retornando true se dois
produtos possuem o mesmo código de barras;
• Na classe Loja, implemente um procedimento de busca que, dado um produto indique em
que posição do vetor de produtos (estoque) se encontra o produto especificado ou
imprima que o mesmo não foi encontrado;

Evite ao máximo repetição de código desnecessário. O programa deve permitir realizar
operações:

• Listar informações da loja, tais como nome e estoque;
• Editar Informações de algum item em estoque;
• Inserir e remover algum item do estoque;
```
